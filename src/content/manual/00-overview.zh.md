# Ventage 是什么 · 解决什么问题

> 5 分钟读完，理解整个平台的定位和模块组合。

---

## 一句话定位

**Ventage = 给个人和小团队用的「机构级 AI 量化研究终端」。**

把对冲基金里几百万美金的工作流（多源情报融合 + 多智能体研究 + 因子研究 + 形态识别 + 回测）压缩成单个用户就能用的工具。

---

| ------------------------------------------------------- | ---------------------------------------------------- |

## 三大核心引擎

### 1. 多状态自适应交易系统 Trading System v2

**六维度协同评分**：

| 维度         | 引擎                                             | 评分占比 |
| ------------ | ------------------------------------------------ | -------: |
| 市场状态     | 状态分类器（6 态：趋势 / 震荡 / 反转 / 事件...） |      25% |
| 价格结构     | 蔡森 12 形态识别 + 等幅满足计算                  |       8% |
| 动量         | RSI / MACD / EMA13-34-55                         |      12% |
| **成交量**   | 成交量引擎（节奏 / 量价 / 突破质量 / 衰竭）      |  **18%** |
| **筹码结构** | 筹码结构（成交量分布 + 高/低频区 + 成本迁移）    |  **22%** |
| 风报比       | 交易管理器（4 类退出 + 仓位算法）                |      15% |

最终输出 0-100 综合分 + **A/B/C 等级**。

### 2. 因子研究系统

基于《因子投资：方法与实践》（石川等，2020）方法论：

- **横截面排序**：全市场按因子值排序，分位数收益对比
- **Fama-MacBeth 回归**：测试因子是否真被定价（Newey-West HAC 调整）
- **PIT 回测**：每月末重新筛选，零前瞻偏差
- **多策略组合**：等权多空对冲，AQR Style Premia 方法
- **质量分桶**：验证 `pattern_quality_score` 是否真有超额收益

### 3. AI 多角色分析

7 个虚拟分析师（基本面 / 技术 / 情绪 / 新闻 / 多头 / 空头 / 风险经理）+ 交易员决策，单次 GPT-4o 调用模拟出投资委员会辩论。

输出：买入 / 持有 / 卖出 + 信心等级 + 入场区间 + 止损位 + 第一目标价。

## 设计原则

Ventage 的所有功能遵循 4 条铁律：

1. **诚实分析** — 即使发现回测胜率低于预期，也如实展示，不修饰
2. **可解释** — 拒绝纯黑盒。每个评分都能追溯到具体子项
3. **时点对齐 (PIT)** — 任何回测都必须使用当时能拿到的数据，无前瞻偏差
4. **零幻觉** — AI 只做归纳分析，所有数字由代码计算后传给 AI
